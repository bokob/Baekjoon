# 풀이
층수 = N % H  
여기서 중요한 것이 N이 H로 나눠지는 경우와 나눠지지 않는 경우로 나눠야 한다.    
1. N % H == 0인 경우  
    * 방번호 = N / H  
2. N % H != 0인 경우  
    * 방번호 = N / H + 1

방번호는 10보다 작으면 '0'을 붙여주고 아니면 그냥 사용한다.  
문자열로 바꾸어서 연결하면 끝이다.  

# 느낀점  
배열 이용해서 푸는 줄 알고 풀다가 규칙이 보였다.  
N이 H로 나눠지는 경우와 나눠지지 않는 경우를 생각하지 않고 풀어서 한 번 틀렸다.  
다음부터 예외적인 상황을 생각해보자.
