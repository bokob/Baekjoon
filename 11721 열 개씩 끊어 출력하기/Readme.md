# 풀다가 알게 된 점
1.
    - getline()은 <string>에 정의되어있음
    
    - cin.getline(char 문자열, 문자열 길이)
    cin.getline(char *s, streamsize n, char delim);  ( delim(제한자) 직전까지 읽어서 저장한다. ) 
    별도로 지정하지 않으면 개행문자로 인식한다. 
    
    - getline(cin,(string 변수))로 한 줄에 다 입력받을 수 있다.(저번에 무지성으로 사용함)
